Chess AI Bot is a Python-based chess engine designed to play, learn, and improve through self-play. Built with python-chess and pygame, the bot features advanced evaluation functions, opening book integration, and reinforcement learning techniques inspired by modern chess engines like AlphaZero.

Features

Human vs AI & AI vs AI – Play against the bot or watch it train itself.

Opening Book Support – Implements standard and custom opening books to start games strongly.

Self-Play Training – The bot improves by playing against itself, tuning evaluation parameters automatically.

Customizable Evaluation – Piece values and positional bonuses are adjustable and learnable.

Alpha-Beta Minimax Search – Efficient search with transposition tables and pruning.

Visual Board – Pygame interface displaying the current board, moves, and highlights.

Learning from Games – Tracks winning sequences to update opening strategies.
